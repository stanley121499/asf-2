## Instructions for Creating Posts

1. **Click the Image Link** for each post
2. **Download your chosen image** from Pexels (free)
3. **Upload image** to post folder via your admin panel
4. **Copy field values** from the table above
5. **Paste into the post editor**:
   - Name field
   - Caption textarea (full text)
   - Select Photo Size dropdown
   - Select Caption Position dropdown
   - Select Font Family dropdown
   - Enter CTA Text
6. **Save the post**
