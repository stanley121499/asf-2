## How to Download and Use

### Option 1: Manual Download (Recommended for Quality Control)
1. Click each URL above
2. Download high-resolution images (free)
3. Rename files to match product (e.g., `turmeric-powder-1.jpg`)
4. Upload to Supabase Storage via admin panel
5. Link images to products in the admin UI

### Option 2: Automated (Advanced)
Create a script to:
1. Fetch images from Unsplash API (requires API key - free)
2. Download and resize images
3. Upload to Supabase Storage
4. Create product_media records

## Recommended Image Specs
- Format: JPG or PNG
- Size: 1200x1200px (square) or 1200x800px (landscape)
- Quality: High (80-90% compression)
- Background: White or natural/wooden surface
- Style: Clean product photography
